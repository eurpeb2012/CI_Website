🔒 Security

  • XSS Prevention — _escapeHtml() applied to all user-generated content (journal, forum, reviews, moderation)
  • SQL Injection — Safe; all queries use Supabase parameterized methods
  • Row-Level Security — Enabled on all 21 tables with 30+ policies
  • Stripe Webhooks — HMAC-SHA256 signature verification with constant-time comparison
  • Rate Limiting — Per-IP limits on all API endpoints
  • Admin Auth — Session + role check before rendering admin panel
  • Secrets Management — All API keys stored as Cloudflare encrypted secrets
  • File Upload Validation — 2MB max, JPEG/PNG/WebP only

🧪 Test Guide

Getting Started

Step 1: Open the app at healing-garden-3w5.pages.dev
Step 2: Tap the profile icon → Log in with the demo account
Step 3: Switch language (JP/EN) via Settings if needed

Test Flow: Book a Session

1. Home → Browse therapists or use Search
2. Tap a therapist → View profile → Select a session
3. Pick a date and time slot on the calendar
4. Agree to cancellation policy → Confirm booking
5. Redirected to Stripe Checkout → Use test card below
6. After payment → Redirected back to success page with points earned

Test Flow: Gift Card

1. Home → Gift Cards
2. Select amount → Enter recipient email → Add message (optional)
3. Send → Stripe Checkout → Pay with test card
4. Recipient gets email with gift card code
5. To redeem: Gift Cards page → Enter code → Apply

Test Flow: Video Call

1. Go to any therapist's chat (Profile → Messages → Select conversation)
2. Tap the video call button (📹) in the chat action bar
3. Daily.co video room opens in embedded iframe
4. End call → Returns to chat, duration logged

Test Flow: Retreat Booking

1. Home → Retreats → Select a retreat
2. View details → Book Now
3. Redirected to Stripe Checkout → Pay with test card

Test Flow: Admin Panel

1. Open admin panel
2. Dashboard shows aggregate stats
3. Navigate sidebar: Therapists, Users, Bookings, Revenue, Moderation
4. System Health → Check DB latency and Stripe status
5. Audit Log → Filter by event type
