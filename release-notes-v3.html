v3.1では、ユーザーフィードバック（3/8）に基づき検索カテゴリの再構成、リトリート専用ページ、掲示板を追加しました。v3.0の22の改善に加え、5つの新しい変更を実装しています。 v3.1 adds restructured search categories, a dedicated Retreats section, and a community Message Board based on user feedback (3/8). These 5 changes build on top of the 22 improvements from v3.0.

v3.1 フィードバック対応 (3/8) v3.1 Feedback Changes (Mar 8)

ユーザーレビューに基づく改善 Improvements based on user review session
CHANGED
🔀

検索カテゴリの再構成 Search Categories Restructured

カテゴリを3つに集約：身体の癒し、心の癒し、楽しい体験。占いとリトリートは「楽しい体験」のサブカテゴリに。条件検索と気持ち検索の両方に反映。 Reduced to 3 main categories: Physical Relief, Mental Relief, Playful Interaction. Fortune Telling and Retreat are now subcategories of Playful. Applied to both criteria and feeling searches.

CHANGED
💴

価格フィルターの更新 Pricing Filter Updated

価格帯の区切りを¥25,000から¥35,000に変更。新しい区分：〜¥8,000 / 〜¥15,000 / 〜¥35,000 / ¥35,000+ Price breakpoint changed from ¥25,000 to ¥35,000. New tiers: up to ¥8K / ¥15K / ¥35K / ¥35K+

CHANGED
🎭

気持ち選択の並び替え Feeling Options Reordered

左側にネガティブな気持ち（ストレス、不安、孤独、疲れ）、右側にポジティブな気持ち（変化、未来、パートナー、仕事）。「新しい体験をしたい」は最後に配置。 Left side: negative/stressed feelings. Right side: positive/aspirational. "Looking for something new" moved to last position.

NEW
🏕️

リトリート専用ページ Dedicated Retreats Section

世界中のウェルネスリトリートを閲覧・予約できる新ページ。バリ島、ハワイ、インド、箱根の4つのリトリートを掲載。セラピストとは独立した提供者。航空券は含まれません。 New page to browse and book wellness retreats worldwide. 4 retreats: Bali, Hawaii, India, Hakone. Independent providers (not therapists). Airfare not included.

NEW
💬

掲示板 / フォーラム Message Board / Forum

コミュニティ掲示板を追加。トピックの投稿と返信が可能（ログイン必要）。リトリート、アロマ、瞑想など3つのサンプルトピック付き。コミュニティガイドライン表示。 Community message board added. Post topics and replies (login required). 3 sample threads on retreats, aromatherapy, and meditation. Community guidelines shown.
