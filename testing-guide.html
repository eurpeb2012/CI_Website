Healing Garden v2

Testing Walkthrough Guide

Open the App → Open Admin →
The main app is best viewed on mobile (375px / iPhone). The admin panel is best viewed at desktop width (960px+). All 11 test flows below cover 85 checkpoints across the full app and admin panel.

1 Anonymous Browsing

The app should be fully browsable without logging in.

  1. Landing Page — You see the Healing Garden logo, title, and two buttons
  2. Tap "Find a Session" (or “セッションを探す”)
  3. You see two search options: By Criteria and By Feeling

Path A: Search by Feeling

  1. Tap "Search by Feeling" → pick any mood (e.g., “Stressed”)
  2. Pick a category (e.g., “Mental Relief”)
  3. Pick a delivery method (e.g., “Video Call”)
  4. You see results — therapists sorted by tier (Premium first), each has a tier badge
  5. Tap any therapist card to view their full profile

Path B: Search by Criteria

  1. Tap "Search by Criteria"
  2. Set filters (type, method, price) and tap Search
  3. Browse results and tap into a profile

On the Therapist Profile, check:

  • Tier badge (🌱 Free / 🌿 Standard / 🌳 Premium) and founding member star (⭐ Founding) where applicable
  • Sessions listed with prices, duration, delivery tags
  • Calendar availability
  • Review tabs — tap between “Reviews from Clients” and “Reviews from Therapists”
  • Suggested Therapists section at the bottom (on Hanako, Mitsuki, Sakura, Ken)

2 Auth Gate (Signup Required)

While still anonymous:

  1. On any therapist profile, tap "Book This Session" or "Send Message"
  2. You should be redirected to the Signup page with a notice that an account is required
  3. You have two options:
    • Fill in name + email and tap Sign Up
    • Or tap "⚡ Demo Login" for instant access as a test user
  4. After signing up/logging in, you should be automatically redirected back to the action you were trying to take (booking or chat)

3 Logged-In Experience

After logging in (use Demo Login for the quickest path):

Booking

  1. Navigate to a therapist → tap "Book This Session"
  2. Review booking summary (session, therapist, date, price)
  3. Check the cancellation policy checkbox
  4. Tap "Confirm & Pay" → see success screen

Chat

  1. On a therapist profile, tap "Send Message"
  2. You see the chat screen with mock conversation bubbles
  3. Note the info bar (“Messages are available for 3 days”)
  4. Tap "Start Video Call" at the top

Video Call

  1. You see a dark full-screen layout with:
    • Remote video placeholder (therapist avatar + name)
    • Small self-view box (top right)
    • Control buttons: Mute, Camera, End, BG Blur
  2. Tap the control buttons — they toggle active state (visual feedback)
  3. Tap End to return to chat

Write a Review

  1. On a therapist profile, scroll to Reviews section
  2. Tap "Write a Review"
  3. Tap stars to set rating, write a comment, submit

4 User Profile

  1. Tap Profile in the bottom nav
  2. Check these sections:
    • Booking History — 5 bookings (3 completed, 2 upcoming) with status badges
    • Your Reviews — 3 reviews you’ve written (tappable, links to therapist)
    • Reviews from Therapists — 3 reviews therapists wrote about you
  3. Tap Settings → see theme selector
  4. Tap Log Out → profile returns to anonymous state

5 Theme Switching

  1. Go to Profile → Settings
  2. You see 3 theme cards with color swatches:
    • 🌸 Spring — soft pastel greens (default)
    • 🌻 Summer — warm amber/orange
    • 🌙 Evening — cool indigo/purple
  3. Tap each theme — the entire app updates instantly (header, buttons, cards, backgrounds)
  4. One theme will show “(suggested)” based on time of day
  5. Theme persists across page reloads

6 Language Toggle

  1. Tap the EN/JP button in the top-right header
  2. All text switches between Japanese and English
  3. Works on every screen — try switching while on different pages

7 Therapist Dashboard

  1. Go to Profile → "Switch to Therapist Mode"
  2. The bottom nav changes to: Dashboard / Sessions / Profile
  3. Dashboard shows: booking count, rating, monthly earnings

Navigate all 7 sub-screens:

8 Therapist Apply Form

  1. From the landing page, tap "Apply as Therapist"
  2. Check the 3-tier plan cards (Free / Standard / Premium) with feature lists
  3. Note the platform fee info box (9% per transaction)
  4. Fill in fields and tap Submit → success screen

9 Referral Landing

  1. Navigate directly to: #/referral/HANAKO2026
  2. You see a referral landing page showing Hanako Yamada as the referrer
  3. Tap “Sign Up Now” → goes to apply form
  4. Try other codes: MITSUKI2026, KEN2026, SAKURA2026

10 Edge Cases

  1. Try navigating to a non-existent route (e.g., #/nonexistent) → should show landing page
  2. Try navigating to a non-existent therapist (e.g., #/therapist/999) → should redirect to search
  3. Try booking without checking the agreement → button stays disabled
  4. Try submitting a review with 0 stars → should not submit
  5. Refresh the page at any point → theme, auth state, and therapist mode should all persist

11 Admin Panel

The admin panel is a separate site for platform operators. Open admin.html from the project root.

Best viewed at desktop width (960px+). The admin layout uses a sidebar navigation instead of bottom nav.
  1. Admin Dashboard — You land on #/admin showing platform stats grid (therapists, users, bookings, revenue, fees), quick links, and recent activity feed

Therapist Management

  1. Click Therapists in the sidebar → table of 8 therapists with tier badges, status, founding star
  2. Use the tier and status filters → table updates
  3. Click View on any therapist → detail page with profile, tier management, booking window control, stats
  4. Change the tier dropdown or booking window (14/30/60/90 days) → “Saved” toast appears

User Management

  1. Click Users → table of 20 users with email, join date, bookings count, status
  2. Type in the search box → table filters in real-time by name or email
  3. Click View on any user → detail page with profile and booking history
  4. Change the status dropdown (Active/Suspended) → “Saved” toast

Bookings

  1. Click Bookings → table of 18 bookings with ID, date, client, therapist, session, amount, status
  2. Use the status filter (All / Upcoming / Completed / Cancelled) → table and totals row update
  3. Click a therapist name → navigates to their detail page

Revenue & Analytics

  1. Click Revenue → stats grid showing gross revenue, platform fees (9%), referral payouts (2%), net income
  2. Check the CSS bar chart showing 5 months of revenue trend
  3. Check the per-therapist revenue table with links to therapist details
  4. Check the fee structure summary card

Content Moderation

  1. Click Moderation → pending queue (3 items) and resolved section (2 items)
  2. Each card shows type badge (Review/Report), content, reporter, target therapist, reason
  3. Click Approve, Remove, or Dismiss on a pending item → item moves to resolved section

Referral Tracking

  1. Click Referrals → stats (total referred, total commission), all referral codes table
  2. Click Disable on an active code → status changes, button becomes “Enable”
  3. Check the referral chain visualization showing who referred whom

Calendar Settings

  1. Click Calendar Settings → global max days, per-tier defaults, per-therapist overrides, blackout dates
  2. Change any dropdown (global, tier, therapist) → “Saved” toast
  3. Click × to remove a blackout date, use the date picker + “Add” to add one

Admin Cross-Cutting

  1. Click the theme dropdown in the admin header → all screens update (Spring/Summer/Evening)
  2. Click the language toggle (EN/JP) in the sidebar footer → all text switches
  3. Test at desktop width (960px+) — sidebar visible, wide tables
  4. Test at tablet width (768px) — sidebar hidden, hamburger menu appears
  5. Click “Back to Site” in the sidebar → returns to main app
